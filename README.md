# 앱소개
로파는 의류를 판매하는 쇼핑몰 앱입니다.

로파는 개성있는 크리에이터들이 상품을 보다 활용성 있게 홍보하고, 

크리에이터들의 홍보를 통해 판매가 이루어졌을 때, 

상점에서 마일리지를 제공하는 서비스가 있습니다.

때문에 구매자에게는 제품의 입체감을, 판매자에게는 상품의 활성도를 제공합니다.
